### 1. OSI Model (7 Layers)
- Layer 7: Application (HTTP, SSH, etc.)
- Layer 4: Transport (TCP/UDP, ports)
- Layer 3: Network (IP addressing and routing)
- Layer 2: Data Link (MAC address)
- Layer 1: Physical (cables, WiFi)

For Cloud Security, Layer 3, 4, and 7 are most important.

### 2. IP Addressing & CIDR
- Every device has an IP address.
- CIDR notation tells how many IPs are in a network.
- Example: /24 = 256 addresses, /27 = 32 addresses.

### 3. Important Linux Networking Commands
- `ip addr show` → Shows IP address and network interfaces
- `ip route` → Shows routing table
- `ping 8.8.8.8` → Test connectivity
- `ping google.com` → Test DNS
- `ss -tulnp` → Show listening ports
